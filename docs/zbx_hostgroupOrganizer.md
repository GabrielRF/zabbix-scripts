# zbx\_hostgroupOrganizer.py


# Usage:
```sh
usage: zbx_hostgroupOrganizer.py [-h] [--url URL] [--user USER]
                                 [--password PASSWORD] [--no-verbose]
                                 [--verbose] [--loglevel LOGLEVEL]

Creates a hostgroup for each proxy, and adds all hosts monitored by it. Also,
interacts with all hosts in Operacao organizing it.

optional arguments:
  -h, --help           show this help message and exit
  --url URL            Zabbix server address
  --user USER          Zabbix user
  --password PASSWORD  Zabbix password
  --no-verbose         Dont show any logs on screen
  --verbose
  --loglevel LOGLEVEL  Debug level. DEBUG/INFO/WARNING/ERROR/CRITICAL
```
